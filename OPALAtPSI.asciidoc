= Compile OPAL

'''

== Load Modules
The modules required are:

```
1) gcc/5.4.0        4) hdf5/1.8.18 
2) openmpi/1.10.4   5) gsl/2.2.1        7) H5hut/2.0.0rc3
3) boost/1.62.0     6) cmake/3.4.1
```

'''
'''

== Get OPAL source

The OPAL source can be obtained from git@gitlab.psi.ch:OPAL/src.git

```
mkdir OPAL_repo
cd OPAL_repo
git clone git@gitlab.psi.ch:OPAL/src.git
```
At the of the cloning process, the source directory is created and filled with the master version of the repository. 

To get a specific branch, for example OPAL/1.6, do:

```
cd src
git checkout OPAL-1.6
```

'''
'''

== Compile OPAL

Inside the OPAL_repo directory create another directory where to compile OPAL. 

```
mkdir build
cd build
cmake ../src/
make -j n
```
